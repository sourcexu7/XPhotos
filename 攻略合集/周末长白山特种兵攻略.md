+ **吃饭：沈阳西塔老街-****<font style="color:#D22D8D;">美里·朝鲜族烤串</font>**
+ **宿 ****<font style="color:#601BDE;">沈阳北-兰朵高奢定制酒店</font>**



**<font style="color:#1DC0C9;">day2</font>**

**<font style="color:#01B2BC;">长白山</font>**

+ **出发	 ****<font style="color:#DF2A3F;">9:25-11:43 </font>****沈阳北 => 长白山**
    - 坐高铁G8135** 12月27日 ****<font style="color:#DF2A3F;">9:25-11:43</font>**** ￥173**
    - **11:50 到达长白山西**

**<font style="color:#01B2BC;">西坡检票中心</font>**

+ **出发	 12:00-13:10 长白山 => 雪岭**
    - **包车大概一小时到达**。

**<font style="color:#01B2BC;">雪岭</font>**

+ **13:10-16:20 雪岭游玩**
    - 雪地摩托上山
    - 雪地也非常厚，非常适合在雪地里打滚拍照
    - 日落后立马去马拉犁耙排队下山

**<font style="color:#01B2BC;">返程干饭</font>**

+ **16:30-17:40 雪岭 => 二道白河镇**
    - 路上联系老板提前炖上到了直接开吃节省时间
    - 二道白河镇吃铁锅炖大鹅-**<font style="color:#D22D8D;">林海雪缘铁锅炖</font>**

**<font style="color:#01B2BC;">蓝景温泉</font>**

+ **19:00-19:30  二道白河镇 => 蓝景温泉**
    - 天然室外温泉
    - 室外有很多类型的温泉，要自备泳裤，现场60块钱一条

**<font style="color:#01B2BC;">返程</font>**

+ **21:00-21:30 回二道白河镇**
+ **21:40酒店入住**
    - **<font style="color:#601BDE;">艾菲酒店（二道白河镇店）</font>**



**<font style="color:#DF2A3F;">day2</font>**

**<font style="color:#DF2A3F;">水色漂流</font>**

+ **出发	 5:30出门**
    - 7:50小沙河漂流
    - 6:00-7:10漂流，水不急，基本不会打湿衣服。这个时段有晨雾，有雾凇，且不需要排队
    - 7:10在结束的地方拍照，岸边雾凇非常明显，光线也很好，非常适合拍照
    - 建议早点去晚一分钟去队伍长一段

**<font style="color:#DF2A3F;">恩都里</font>**

+ **7:30-8:00  水色 => 恩都里**
+ **8:30-10:00**
    - **恩都里打卡云顶天宫、雪王军团 **
+ **10:00-10:30 恩都里 => 二道白河镇**

**<font style="color:#DF2A3F;">吃饭</font>**

+ **10:30-11:50 司机找的一家****<font style="color:#D22D8D;">当地的东北菜</font>**
    - **锅包肉好吃要吃**

**<font style="color:#DF2A3F;">返程</font>**

+ **11:50-12:10 二道白河镇 => 长白山站**
+ **<font style="color:#DF2A3F;">12:21-14:25</font>**** 长白山 => 沈阳**
+ **15:00-16:10 沈阳按摩**
+ **16:10-17:10 沈阳到桃仙机场**
+ **<font style="color:#DF2A3F;">18:55-21:05 </font>****沈阳飞回徐州**
